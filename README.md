# 📝 To-Do List Application

A simple and interactive To-Do List application built using **HTML, CSS, and JavaScript**. This project allows users to add tasks, mark them as completed, and delete tasks when finished. It also includes a dashboard that tracks task progress.

<a href='https://shreya-103.github.io/to-do-list'> try it here </a>

## 🚀 Features

* Add new tasks dynamically.
* Mark tasks as completed using a checkbox.
* Delete tasks with a single click.
* Prevent empty tasks from being added.
* Dashboard displaying:

  * Total number of tasks.
  * Number of completed tasks.
  * Motivational compliments based on progress.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## 📋 How It Works

### Adding a Task

1. Enter a task in the input field.
2. Click the **Add Task** button.
3. A new task item is created and displayed on the page.

### Completing a Task

* Click the checkbox beside a task.
* The task becomes semi-transparent, indicating completion.

### Deleting a Task

* Click the delete icon beside a task.
* The task is immediately removed from the list.

### Dashboard

The dashboard updates automatically to display:

* Total Tasks
* Completed Tasks
* Progress Compliment

Examples:

* **Outstanding!** → All tasks completed.
* **Keep it up!** → Some tasks completed.
